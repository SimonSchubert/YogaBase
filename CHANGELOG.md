# Changelog

## v1.1.1 — 2026-04-24

First production release.

### Features
- Native multiplatform app built with Kotlin Multiplatform (Android, iOS, Desktop, Web)
- 48 yoga poses across 25 guided session categories
- Per-pose timer with Swift / Balanced / Endurance difficulty multipliers
- Voice guidance with session start, pose-switch, and congratulations cues
- Streak and progress tracking with local storage
- XP and level progression with milestone titles
- Full-screen workout mode

### Improvements
- Refined exercise UI, styling, and pose data
- Improved bundled audio output

### Fixes
- Pose title positioning
- Tick indicator position and "day / days" pluralization
- Exercise end celebration and live progress display
